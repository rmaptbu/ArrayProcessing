writeAttributes

Write attributes to a k-Wave HDF5 file

Syntax

writeAttributes(filename)
writeAttributes(filename, file_description)

Description

writeAttributes writes a set of additional attributes to the HDF5 file specified by the user. These attributes specify the file version, k-Wave version, type, and so on, and are required by the k-Wave C++ simulation code. If the file_description is not specified, a description is generated based on computer username and MATLAB version.

Inputs

filename

name of HDF5 file to write matrix to

file_description

custom file description
