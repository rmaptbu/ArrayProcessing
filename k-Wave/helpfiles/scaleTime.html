scaleTime

Convert seconds to hours, minutes, and seconds

Syntax

time = scaleTime(seconds)

Description

scaleTime converts an integer number of seconds into hours, minutes, and seconds, and returns a string with this information.

Inputs

seconds

number of seconds

Outputs

x_sc

string of scaled time
