interpftn

Resample data using Fourier interpolation

Syntax

y = interpftn(x, sz)

Description

interpftn resamples an N-D matrix to the size given in sz using Fourier interpolation.

Inputs

x

matrix to interpolate

sz

new size

Outputs

y

resampled matrix
